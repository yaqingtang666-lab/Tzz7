## 一、核心技术架构

### 1.1 系统整体架构

本系统基于 Taichi 框架构建，采用数据场（Field）与核函数

### 1.2 弹簧模型架构

本系统实现了三种类型的弹簧：

| 弹簧类型 | 连接方式 | 劲度系数 | 物理作用 |
|---------|---------|---------|---------|
| **结构弹簧 (Structural)** | 水平/垂直相邻质点 | k_s | 抵抗拉伸与压缩，维持布料基本形状 |
| **剪切弹簧 (Shear)** | 对角相邻质点 | k_sh | 抵抗剪切变形，防止布料歪斜 |
| **弯曲弹簧 (Bending)** | 间隔一个质点的连接 | k_b | 抵抗弯曲，使布料更加平整 |

弹簧数据场设计：
- `spring_pairs`: 存储弹簧连接的两个质点索引
- `spring_lengths`: 存储弹簧的原长（初始长度）
- `spring_stiffness`: 存储每个弹簧的劲度系数，支持不同类型弹簧使用不同参数

### 1.3 碰撞检测架构

球体碰撞采用距离检测法：
1. 计算每个质点到球心的距离
2. 若距离小于球半径，则发生碰撞
3. 将质点位置推到球面上
4. 根据碰撞法向反射速度（保留 20% 法向速度分量）

### 2.2 阻尼系数对比

**实验条件**：结构弹簧 k_s=10000，剪切弹簧 k_sh=5000，弯曲弹簧 k_b=1000

| 阻尼系数 k_d | 观察到的现象 |
|-------------|-------------|
| **k_d = 1.0** | 系统能量衰减较慢，布料在重力作用下下落时会出现明显的振荡和晃动，最终缓慢趋于静止。振荡幅度大，稳定时间长。 |
| **k_d = 10.0** | 系统能量快速衰减，布料下落时振荡幅度明显减小，快速趋于平衡状态。布料形态更加"僵硬"，运动过程更加平滑。<img width="320" height="343" alt="b6mNiuR5_converted" src="https://github.com/user-attachments/assets/271c072f-cb18-455f-822a-19adfc07dfcc" />|

**物理原因**：阻尼力 f_d = -k_d * v 与速度成正比，增大阻尼系数会加速系统动能向热能的转化，从而更快耗散能量，抑制振荡。

### 2.3 弯曲弹簧劲度系数对比

**实验条件**：结构弹簧 k_s=10000，剪切弹簧 k_sh=5000，阻尼系数 k_d=1.0

| **k_b = 5000** | 布料褶皱明显减少，弯曲刚度增加。布料更加平整，与球体接触时形变较小。<img width="640" height="686" alt="NyPJfjmu_converted" src="https://github.com/user-attachments/assets/c1745744-98bf-479a-a219-c05e225f7810" /> |

**物理原因**：弯曲弹簧连接间隔一个质点的节点，增大 k_b 会增加布料的弯曲抗力，使布料倾向于保持平整状态，减少过度弯曲。

### 2.4 结构弹簧劲度系数对比

**实验条件**：剪切弹簧 k_sh=5000，弯曲弹簧 k_b=1000，阻尼系数 k_d=1.0

|-----------------|-------------|
| **k_s = 10000** | 布料在重力作用下会适度拉伸，质点间距增大但保持整体形状。与球体碰撞时会产生明显的凹陷变形。 |
| **k_s = 50000** | 结构刚度大幅增加，布料几乎不发生拉伸变形。布料表现为更硬的薄板状，与球体接触时变形极小。<img width="480" height="515" alt="235Ak3eI_converted" src="https://github.com/user-attachments/assets/b9fc2447-7907-4e9b-b858-b8ff53c5ea42" /> |

## 三、实验原理

### 3.1 三种弹簧


1. **结构弹簧**提供基本的骨架支撑，防止质点间距过大
2. **剪切弹簧**提供斜向支撑，防止布料被"撕碎"成菱形
3. **弯曲弹簧**提供抗弯曲能力，使布料呈现自然的悬垂特性

仅使用结构弹簧的布料会出现过度柔软、容易扭曲的问题；加入剪切弹簧后布料具有一定的抗剪切能力；弯曲弹簧则使布料呈现出更加真实的褶皱和悬垂效果。

### 3.2 碰撞处理


```python
d = pos[i] - center          # 质点到球心向量
dist = d.norm()              # 距离
if dist < radius:            # 碰撞检测
    normal = d / dist         # 碰撞法向
    pos[i] = center + normal * radius  # 位置修正
    vel[i] = vel[i] - vel[i].dot(normal) * normal * 0.8  # 速度反射
```

## 四、实验总结

通过本实验，我观察到：

| 参数 | 增大值的效果 |
|-----|-------------|
| 阻尼系数 k_d | 系统更快稳定，振荡减少，运动更平滑 |
| 弯曲弹簧 k_b | 布料更平整，褶皱减少，抗弯曲能力增强 |
| 结构弹簧 k_s | 布料更难拉伸，形状更刚性，形变减少 |

这些参数的综合调整可以实现不同材质的布料效果。
